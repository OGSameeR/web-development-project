# 🕊️ Naye Pankh Foundation Website Template

A modern, highly responsive, and visually engaging multi-page website template designed for the **Naye Pankh Foundation** NGO. This project was developed focusing on clean semantic markup, fluid layouts, and performance-driven CSS animations to ensure an optimal user experience across all devices.

---

## ✨ Features

- **Multi-Section/Page Flow:** Seamlessly navigates through Home, About Us, Impact/Stats, and Contact/Donate sections.
- **100% Responsive Design:** Built using **CSS Flexbox and Grid** to ensure flawless rendering on mobiles, tablets, and desktops.
- **Engaging Animations:** Subtle, custom CSS keyframe animations (fade-in updates, floating logo effect) to improve user retention without degrading performance.
- **Interactive UI Components:**
  - Dynamic mobile hamburger menu powered by JavaScript.
  - Interactive donation component with pre-set amount selections.
  - Smooth-scroll behavior enabled across all internal section identifiers.

---

## 🛠️ Tech Stack & Concepts Demonstrated

- **Structure:** HTML5 (Semantic elements like `<nav>`, `<section>`, `<footer>` for SEO optimization).
- **Styling & Layout:** CSS3 (Custom properties/variables, Grid, Flexbox, Media Queries).
- **Interactivity:** Vanilla JavaScript (DOM manipulation, event listeners, scroll-tracking logic).
- **Icons:** FontAwesome v6.4.0 integration.
